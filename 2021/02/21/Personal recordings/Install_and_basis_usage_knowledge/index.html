docker

Using docker to install enviroments and run some codings. we can find some images in docker hub.

Basic operations

volume

  • “docker volume create”: create a volume

  • “docker volume ls”: show all volumes

  • “docker volume inspect my-vol”: show the detail info. of volume my-vol.

  • “docker volume rm my-vol”: delete a volume.

Docker compose

  • Firstly, we need a “docker-compose.yml” file

  • Then, excute “docker-compose up (-d)”, where “docker-compose up” means run the procedure based on “docker-compose.yml”, adding “-d” means running in the background.

Quantaxis

Quantaxis is a tool to monitor stock. We can take communications with other through platform and find some instruction in documents.

QAStrategy

QAStrategy is a basic model of Quantaxis, which can deploy through docker. The instruction is here and here

  • Download the “docker-compose.yml” file of Quantaxis

  • Establish the volumes

  • Upgrade “docker-compose.yml” by “docker-compose pull”

  • Run the QUANTAIS service by excuting “docker-compose up -d”.

  • Open the 8888 port by “127.0.0.1:8888” to save stock_list. (password is “quanatxis”)

  • click the terminal and type “bash”, “quantaxis”, “save stock_list”

  • restart the “qaweb” in the docker.

  • Important ports

    • 27017 mongodb
    • 8888 jupyter
    • 8010 quantaxis_webserver
    • 81 quantaxis_community
    • 61208 system monitoring
    • 15672 qa-eventmq
